# Ahmed Abdelwahed - Personal Website

Welcome to the repository for my personal website! This project serves as my digital portfolio, showcasing my background, experience, projects, and articles in the field of Data Engineering and Software Development.

You can view the live website here: [https://ahmedabdelwahed.me](https://ahmedabdelwahed.me/)
## 🚀 Tech Stack

This website is built using modern web development tools:

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Content Management**: Markdown/MDX with `remark` & `rehype` and local JSON files
- **Theming**: Dark and Light mode via `next-themes`

## ✨ Features

- **Responsive Design**: Clean and functional layout across mobile, tablet, and desktop viewports.
- **Dynamic Content**: Portfolio content (blog posts, projects, experiences) is generated automatically from Markdown and JSON files in the `content/` directory.
- **Theme Toggling**: Seamless switching between light and dark modes.
- **Micro-Animations**: Smooth, professional interactions using Framer Motion.
- **Performance Optimized**: Fast page load times and optimized assets using built-in Next.js optimizations.


## 📁 Source Code Structure

- `src/app/`: Next.js App Router pages and layouts.
- `src/components/`: Reusable React components (UI elements, theme toggle, sections).
- `content/`: All site data, including JSON descriptors for the hero section and markdown files for projects, experience, and blog entries.
- `public/`: Static assets such as images and the CV file.
